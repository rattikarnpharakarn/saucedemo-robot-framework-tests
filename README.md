# 🎭 Saucedemo Robot Framework Automation Tests

This repository contains automated functional tests for the [SauceDemo](https://www.saucedemo.com/) website (Swag Labs). The project is built using **Robot Framework** and **SeleniumLibrary**.

## 🚀 Project Overview
The goal of this project is to automate the end-to-end user journey on the SauceDemo site, ensuring that core features work as expected.

### Key Scenarios Covered:
* **Authentication:** Valid/Invalid login and Logout.
* **Product Management:** Browsing products, adding/removing items from the cart.
* **Checkout Process:** Complete flow from cart to order confirmation.
* **Data Validation:** Checking price accuracy and inventory details.

## 🛠 Prerequisites
Ensure you have the following installed on your machine:

1. **Python 3.x**
2. **Robot Framework**
3. **SeleniumLibrary**
4. **Browser Drivers:** (e.g., ChromeDriver for Chrome or GeckoDriver for Firefox)

### Installation
Run the following command to install the required libraries:
```bash
pip install robotframework robotframework-seleniumlibrary
